🏏 TurfTime – Sports Turf Booking Website

    TurfTime is a modern Sports Turf Booking Web Application that allows users to search, view, and book sports turfs easily.
    The platform supports multiple sports like Cricket, Football, and Badminton, and provides features like slot booking, booking history, and dark mode.

🚀 Features

    🔍 Search Turf – Find available sports turfs quickly

    📅 Slot Availability – View available time slots

    🏟 Multiple Sports Support – Cricket, Football, Badminton

    📝 User Registration & Login

    📊 Booking History – Track previously booked slots

    🌙 Dark Mode – Toggle between light and dark themes

    📱 Mobile Responsive Design

    🎨 Modern UI Design
    

🖥️ Technologies Used

    HTML5

    CSS3

    JavaScript

    Responsive Web Design
