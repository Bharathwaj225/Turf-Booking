# 🏏 TurfTime – Sports Turf Booking Website

TurfTime is a modern **Sports Turf Booking Web Application** that allows users to easily search, view, and book sports turfs.
The platform supports multiple sports like **Cricket, Football, and Badminton** and provides a clean, responsive interface for a smooth user experience.

---

## 🚀 Features

* 🔍 **Search Turf** – Find available sports turfs quickly
* 🏏 **Multiple Sports Support** – Cricket, Football, Badminton
* 📅 **Slot Booking** – Reserve available time slots
* 👤 **User Authentication** – Login & Registration system
* 📜 **Booking History** – View previously booked turfs
* 🌙 **Dark Mode** – Toggle between light and dark themes
* 📱 **Responsive Design** – Works on desktop and mobile devices
* 🎨 **Modern UI** – Clean and professional user interface
